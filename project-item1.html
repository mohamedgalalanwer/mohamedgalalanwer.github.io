A Maze Project



Part of the Virtual Reality Nanodegree Program.Place the walls, add lights and materials, and then code a fully functional VR maze game.
You’ll build 3D UI, waypoint-based navigation, clickable objects, and utilize spatial audio.
Players will navigate the maze, collect coins along the way, then find a key that opens a gate to a secret treasure.


Unique Considerations


An important design consideration was to create simulator sickness and choose the perfect way to make movement in VR and make the audio system with GVR audio listener and Sources to make an impressive experience.


Project title:

Built a VR application using Unity and the Google VR SDK where the user explores a maze environment to demonstrate a working knowledge of 2D and 3D UI, waypoint based navigation, procedural animation, interactive objects, spatial audio, particle effects and persistent storage of session data.

This project is about combining various ideas and skills we’ve been practicing throughout the course. They include:


  • 1. Creating new C# scripts in Unity
  • 2. If then, loops, arrays, and other programming constructs
  • 3. Attaching scripts to objects
  • 4. Using the built-in Monobehaviour methods
  • 5. Triggers and Gaze-Based Interaction
  • 6. Creating, moving and animating objects procedurally
  • 7. Familiarization with the Unity documentation
  • 8. Scripting Dynamic UI Objects
  • 9. Debugging
  • 10. The Unity Event System.
  • 11. Managing and Reloading scenes.
  • 12. Controlling particle systems.
  • 13. Create an Audio Clip and playing sounds.
  • 14. Waypoint Navigation System.
  • 15. Profiling scenes for performance.
