Mohamed’s Puzzler Project



As part of my VR nano degree at Udacity, I designed, tested and iterated a mobile VR game (Puzzler). In this game, player solves a puzzle which tests his memory Skills.


Unique Considerations


An important design consideration was to create simulator sickness and choose the perfect way to make movement in VR and make the audio system with GVR audio listener and Sources to make an impressive experience.


Project Title:


introduced: the puzzler is a mobile VR application for a new VR users which challenges them to solve a familiar type of puzzle in a new way, helps users have fun and blow off steam by solving the puzzle and restart the game.




Design Process

Statement of Purpose: Puzzler is a mobile VR game make user challenge his memory by a memorization puzzle on the room he can’t move until solving the puzzle.
